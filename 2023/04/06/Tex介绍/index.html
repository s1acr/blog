简要介绍TEX和 Live Tex安装

TEX

TEX是Donald Ervin Knuth编写的电子排版系统,提供了一套功能强大并且十分灵活的排版语言，它多达 900 多条指令，并且 TeX 有宏功能，用户可以不断地定义自己适用的新命令来扩展 TeX 系统的功能。Tex本身是一种宏语言，而其他基于Tex的二次开发是Tex的宏语言。
TeX 的源程序是用 Pascal 写成的，原因是 Knuth 希望 TeX 尽可能方便地移植到其它的操作系统中去。
TeX 的另一个重要的特征就是它的输出是与设备无关的。TeX 的输出文件称为 DVI 文件，即是Device Independent。一旦 TeX 处理了你的文件，你所得到的 DVI 文件就可以被送到任何输出设备如打印机，屏幕等并且总会得到相同的结果，而这与这些输出设备的限制没有任何关系。

Tex相关

LaTex

Leslie Lamport 开发的 LaTeX 是当今世界上最流行和使用最为广泛的TeX格式。LaTex是基于Tex的排版系统, 是由Tex宏命令组成的宏包.

TeX发行版

将排版引擎,宏包,编辑器,GUI等大量的文件组合打包起来发行, 就组成了TeX发行版.

Texlive：适用于 win 和 Linux 平台。安装包体积较大（4.7GB），自带数量巨大的宏包，也可以用附带的工具安装其它宏包。
Mactex：仅使用于 MacOS，特点同上。
Miktex：适用于所有平台，安装包体积小，排版时按需联网下载所需宏包。

在线LaTex编辑

如果你不想下载软件, 可以使用在线编辑器:

Overleafhttps://cn.overleaf.com/
国内推荐 TeXPagehttps://texpage.com/

下载TeX Live

TeX Live是 TUG (TeX User Group) 发布并维护的的 TeX 系统，可以称得上是TeX的官方系统。我们推荐任何阶段的TeX用户，都尽可能使用 TeX Live，以保持在跨操作系统平台、跨用户的一致性。
下载TeX Livehttps://tug.org/texlive/quickinstall.html

选择相应的系统下载, 我是windows, 选择用installer下载

然后跟着提示安装, 把TexLive自带的TeXworks编辑器也装上

下载很慢, 显示要两个多小时, 耐心等待

下载完CMD输入 tex -v就检测tex是否安装成功

打开TeXwork就可以开始编写了
