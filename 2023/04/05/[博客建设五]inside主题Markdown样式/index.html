代码块

Gist

{% gist gist_id [filename] %}

gist 是github一个实用的子服务， 用来分享代码段 snippets， 需要开代理才能显示

jsfiddle

{% jsfiddle shorttag [tabs] [skin] [width] [height] %}

mathjax是一个用于latex、mathml和ascimath表示法的开源javascript显示引擎.
