Markdown介绍

Markdown是一种轻量级标记语言, 旨在于成为一种易读易写书写格式, 兼容HTML, 广泛应用于文档编写, 已有越来越多的网站支持Markdwon.

Markdown编辑器

推荐VSCode Obsdian 和 Typora, 这里介绍Typora, 现已收费.

编辑模式

typora是Seamless Live Preview, 在普通编辑模式下输入markdown语法会实时预览, 但我一般不在这个模式下书写, 因为自动生成会妨碍我们输入.
ctrl+/切换到源代码模式, 开始书写真正的Markdown.

Typora常用快捷键
ctrl / 切换编辑模式
shift F12 开发者工具
ctrl shift L 显示/隐藏侧边栏

建议记住上面三个
ctrl shift S 另存为
ctrl F 查找
ctrl P 查找文档
ctrl N 新建文件
ctrl shift N 新建面板
